<h2>TL;DR</h2>

```ruby
pry[0]> arrayA = [1, 2, 3,]
pry[1]> arrayB = [4, 5]
pry[2]> arrayA.push(*arrayB)

=> [1, 2, 3, 4, 5]
```

<h3>結論</h3>

<p>push する配列に splat operator (*) なるものをつけてあげるだけで良さそう</p>

```ruby
pry[0]> arrayA = [1, 2, 3,]
pry[1]> arrayB = [4, 5]
pry[2]> arrayA.push(*arrayB)

=> [1, 2, 3, 4, 5]
```
